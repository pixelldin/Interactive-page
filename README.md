## 🚀 Deployment

This project is deployed using **GitHub Pages**.

To deploy:

1. Go to **Settings → Pages**
2. Under **Source**, choose:
    - Branch: `main`
    - Folder: `/docs` *(or `/root` if your files are in root)*
3. Click **Save**

Your page will be live at:
